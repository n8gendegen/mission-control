# Stand up Atlas social profiles + daily cron cadence

## Summary
Establish and automate Atlas social media presence across LinkedIn, X, Instagram, and TikTok by setting up profiles, scheduling content, and implementing daily posting and engagement routines.

## Definition of Done
All four profiles show the new branding + pinned concierge link, initial post batch is drafted/scheduled, cron automation documented + enabled (08:00/14:00 posts, 11:00/20:00 engagement sweeps), runbook updated for future campaigns.

## Acceptance Criteria
- Profile pics/banners/bios updated to the latest creative on LinkedIn/X/IG/TikTok
- Concierge URL pinned in profile + Link in Bio pages
- Post batch: 1 LinkedIn essay, 1 X thread, 1 Instagram carousel, 1 TikTok intro uploaded to drafts with copy + hashtags
- Cron/LaunchAgent or scheduler entries created for the 08:00/14:00 post cadence and 11:00/20:00 engagement sweeps
- Atlas runbook documents login handoff, cron control, and escalation triggers

## Handoff Actions
1. Provide access credentials for all social media accounts to the designated team member.
2. Share the Atlas runbook document with the operations team.
3. Confirm that all cron jobs or scheduler entries are active and logging correctly.
4. Schedule a brief walkthrough of the social media setup and automation with the content and ops teams.
