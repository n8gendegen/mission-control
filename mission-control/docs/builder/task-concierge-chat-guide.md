# Concierge — DeepSeek install chat guide

## Summary
Implement a chat guide within the Concierge interface to assist users with installing DeepSeek, providing step-by-step instructions and troubleshooting tips.

## Definition of Done
undefined

## Acceptance Criteria
- Users can access the chat guide from the Concierge interface.
- The guide provides clear, step-by-step instructions for installing DeepSeek.
- Includes troubleshooting tips for common installation issues.
- Guide is interactive, allowing users to navigate between steps or ask for clarification.
- Content is accurate and up-to-date with the latest DeepSeek installation requirements.
- UI is consistent with the existing Concierge design system.
- Guide is accessible, following WCAG 2.1 AA standards.
- Performance: guide loads within 2 seconds on average network conditions.

## Handoff Actions
1. Review the Concierge UI components and design system for integration consistency.
2. Confirm the latest DeepSeek installation steps and requirements with the product team.
3. Set up a staging environment for testing the chat guide functionality.
4. Schedule a demo with the product owner to validate the implementation against acceptance criteria.
5. Document any assumptions or unknowns (e.g., specific error messages for troubleshooting) for team review.
