# Add one-click critique trigger

## Summary
Implement a one-click critique trigger button on each /content card that initiates the critique automation process. The critique output will be attached in markdown format to the content card, and content owners will be notified upon completion.

## Definition of Done
undefined

## Acceptance Criteria
- A visible button labeled appropriately is present on each /content card.
- Clicking the button triggers the critique automation process without errors.
- The critique output is attached to the content card in markdown format.
- Content owners receive a notification when the critique is ready and attached.
- The feature works consistently across supported browsers and devices.
- Error handling is in place for failed critique automation calls.
- The UI remains responsive during the critique process.

## Handoff Actions
1. Clarify the exact notification method and channels for content owners.
2. Provide details or access to the critique automation API or service endpoint.
3. Confirm the markdown format requirements for the critique attachment.
4. Share any design assets or UI guidelines for the button appearance.
5. Identify content owner determination logic or data source.
